# CLAUDE.md — 삼신학원 웹사이트

## 응답 언어
모든 응답은 한국어로 한다.

## 기술 스택
- 순수 HTML / CSS / JavaScript
- 프레임워크 없음 (React, Vue, Angular 등 사용 금지)
- 외부 라이브러리는 기존에 사용 중인 것만 유지

## 컬러 스킴
- 와인레드: `#8B1A2F` 계열
- 크림: `#FFF8F0` 계열
- 모든 UI 수정 시 이 컬러 스킴을 반드시 유지한다

## 파일 보호
아래 폴더는 절대 수정하거나 삭제하지 않는다:
- `data/`
- `uploads/`
- `.git/`

## 워크플로우
새 기능 추가 또는 큰 수정 시 아래 순서를 따른다:
1. 계획을 먼저 보고한다
2. 사용자 확인을 받는다
3. 확인 후 구현한다

버그 수정 등 소규모 수정은 즉시 진행 가능하다.

## 절대 금지
- API 키, 비밀번호 등 민감 정보 하드코딩 금지
- 개인정보가 포함된 파일 커밋 금지
- 에러와 무관한 파일 수정 금지
- 요청 범위를 임의로 확장하는 리팩토링 금지

## 에이전트 구조
```
orchestrator (팀장)
├── generator    — HTML/CSS/JS 구현 전담
├── debugger     — 에러 추적 및 수정 전담
└── ui-reviewer  — UI/UX 검토 및 수정 전담
```
- orchestrator는 직접 코드를 수정하지 않는다
- 구현/수정은 반드시 담당 에이전트에 위임한다
- 에이전트 파일 위치: `.claude/agents/`
